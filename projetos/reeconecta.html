Sobre o projeto

O ReeConecta é uma plataforma web desenvolvida durante o 2º eixo do curso de Análise e Desenvolvimento de Sistemas da PUC Minas com o objetivo de contribuir para a destinação correta dos resíduos eletroeletrônicos (REEs) e incentivar práticas de economia circular. A solução conecta cidadãos, empresas e cooperativas por meio de um marketplace para comercialização de equipamentos eletrônicos usados, antigos ou com defeito, além de disponibilizar um mapa interativo que permite localizar pontos de coleta e descarte adequados.


O sistema contempla funcionalidades como autenticação de usuários, cadastro de anúncios, gerenciamento de produtos, painel administrativo, pesquisa de itens, geolocalização de pontos de coleta, gerenciamento de perfis e integração entre diferentes perfis de usuários. Durante o desenvolvimento participei da implementação de funcionalidades do sistema, integração entre front-end e back-end, modelagem do banco de dados, testes da aplicação, versionamento utilizando Git e GitHub e publicação da aplicação na Microsoft Azure, aplicando conceitos de engenharia de software, desenvolvimento full stack, APIs REST e metodologias ágeis.

Tecnologias utilizadas
